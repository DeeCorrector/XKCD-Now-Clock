XKCD Now Clock

This is a real-time client side version of http://xkcd.com/now/.
Inspired by this thread on hacker news and this project by Dave Justice.
You can checkout the source here.

DISCLAIMER:
The clock uses UTC and the map's orientation is based on the Greenwich Meridian (roughly England). Therefore England's orientation will always show the correct time but due to scaling issues and things like Daylight Saving time it might be off for other locations.
